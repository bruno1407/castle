                                                RELAIS ET CHÂTEAU

                                                

Sujet : 
Aller sur le site Relais Chateau pour retrouver les +-150 Hôtels restaurant et comparer ces restaurants avec les +-624 restaurant étoilé Michelin
Un deuxième sujet consiste à scraper les prix des hôtels le week-end pour trouver le week-end le moins cher 

Ce projet comporte 2 parties :

-Une première fonction "HotelRestaurantEtoile()" qui retourne :
    -Les restaurants des hotels de Relais-Chateau mais seulement ceux étoilés par le site michelin

-Une deuxième fonction "PrixHotelLeSamedi()" qui retourne :
    -Le nom de l'hôtel
    -l'ID de l'hôtel
    -Les prix des samedis du mois de mars de l'hôtel
    -Et le jour ou le samedi est le moins cher du mois.

Les deux fonctions retournent le résultat dans un fichier JSON situé dans le projet.

Pour lancé le projet : 
    -Allez dans le fichier Projet.js
    -Commentez ou décommettez en bas du fichier la partie que vous voulez voir
    -Lancez le projet grace a la commande : node Projet.js
